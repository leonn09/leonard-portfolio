Introduction to Responsive Design

Responsive design is a method of building websites that work well on all screen sizes â€” from mobile phones to large desktop monitors.


Key benefits:

  • Better user experience across devices
  • Avoids zooming or horizontal scrolling
  • Improves accessibility and SEO
The Viewport Meta Tag

To make your page responsive, add this meta tag inside the <head>:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

What it does:

  • Defines the visible area of your website.
  • Helps scale and fit your layout on all devices.
Media Queries

Media queries allow you to change styles based on screen size or device type.

@media (max-width: 600px) {
   body {
      background-color: lightblue;
   }
}

This means: if the screen width is 600px or less, change the background color.

Relative Units

%, em, rem, vw, vh

Use relative units for scaling content across devices.

  • % - Based on parent element's size
  • em - Based on current element's font size
  • rem - Based on root font size
  • vw - 1% of viewport width
  • vh - 1% of viewport height

Example:

.container {
   width: 80%;
   padding: 2rem;
   font-size: 1.2em;
}
Responsive Images and Flexbox Grid

Images can scale properly using these styles:

img {
   max-width: 100%;
   height: auto;
}


To structure layouts responsively, use Flexbox or Grid:

Flexbox example:

.container {
   display: flex;
   flex-wrap: wrap;
}


Grid example:

.container {
   display: grid;
   grid-template-columns: 1fr 1fr;
}
